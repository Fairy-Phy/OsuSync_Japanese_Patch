# Osu! Sync日本語化パッチ

※パッチって言ってるけどパッチじゃないです。

Syncがv2.18.2から止まって次期バージョンのv3.0(EmberTools)の開発をしているのでおそらく日本語対応バージョンのリリースは相当先になると思います。プルリクエストはすでにマージされていますがこちらでも公開しようと思います。そのほうが翻訳修正を入れるときに便利なので(・∀・)

## ダウンロード&インストール

### ダウンロード

まずは下のリンクから最新バージョンの「バージョン_ja-JP.zip」をダウンロードしてください

**注意!!!! : Source Code (zip or tar.gz) は全然違うのでご注意ください!**

### **[リリース一覧](https://github.com/Fairy-Phy/OsuSync_Japanese_Patch/releases)**

### インストール

zipファイルのダウンロードが終わったらそれを解凍してください

そうすると「ja-JP」フォルダができると思うのでそれをフォルダごと `[Syncがインストールされてるフォルダ] >> [Language]` フォルダに移動させます
(もし **「ja-JP」フォルダ内ではなくではなく解凍した場所にlangファイルが出てしまった場合**は「ja-JP」フォルダを作ってそこに入れてから移動させてください)

移動が完了したらSyncを起動して以下のコマンドを入力します

```
lang ja-JP
restart
```

Sync再起動時に日本語になっていれば成功です!
